## Executive Summary

The `heroes.html` page accurately reflects the high-level concept of seven agent profiles, but several capability claims overshoot the current implementation. The biggest trust risks are:

1. **Social media and CRM integrations are marketed as available today**, yet the code has only dry-run placeholders and the skill docs explicitly mark HubSpot/Salesforce sync and social publishing as roadmap. A visitor expecting working Instagram/Twitter/LinkedIn/Mailchimp or CRM connectors will be disappointed.
2. **NVIDIA integration is overstated.** The Guardian can recognize NVIDIA-style API keys and recommend models from a static catalog, but there is no live NVIDIA API connection or GPU allocation.
3. **"Live payment processing" and "real-time dashboards" are unqualified.** Both require external credentials (`STRIPE_SECRET_KEY`, `GOOGLE_SERVICE_ACCOUNT_JSON`) and, in the dashboard case, the code does not actually provide automatic real-time updates.
4. **Several tool badges (Google Drive, Google Tasks, Notion, PM Suite, Security Dashboard, Encryption, MCP Security) point to non-existent integrations**, which could make the product look more mature than it is.

**Recommendation:** Add clear "requires credentials" or "roadmap" labels throughout the page, remove badges for integrations that are not implemented, and soften language around orchestration, real-time updates, and enterprise security until the corresponding code lands.
