# Investigacion: 10 Ejemplos de Storytelling Web

## Parte 1: Storytelling de Marca en Sitios SaaS (5 ejemplos)

### 1. Stripe — stripe.com
- **Metafora**: Infraestructura global de confianza
- **Tecnica**: Escala progresiva, contador de PIB en vivo, demos interactivas de checkout
- **Arco**: Exposicion (infraestructura del mundo) -> Desarrollo (demos) -> Climax (empresas Fortune 100) -> Resolucion (empieza ahora)
- **Visual**: Wave animation con gradientes organicos (rosa/naranja/azul), GDP counter, bento box UI, tipografia sans-serif geometrica
- **Paleta**: Blanco + gradientes vivos

### 2. Linear — linear.app
- **Metafora**: Evolucion biologica — "A new species of product tool"
- **Tecnica**: Producto como hero (la interfaz real ES el hero), metafora de conduccion autonoma
- **Arco**: Captura (producto funcionando) -> Contexto (nueva especie) -> Problema/Solucion (Intake) -> Estrategia (Plan) -> Ejecucion (Build)
- **Visual**: Dark theme, interfaz real del producto como hero, diagramas 3D isometricos (FIG 0.2, 0.3, 0.4), tipografia serif + sans
- **Paleta**: Dark theme + blanco

### 3. Vercel — vercel.com
- **Metafora**: Infraestructura antropomorfica — "Agentic Infrastructure" que "piensa"
- **Tecnica**: Estructura por capas (Agents -> Apps -> Platforms), minimalismo narrativo
- **Arco**: Concepto (infrastructure that thinks) -> Capa 1 (agents) -> Capa 2 (apps) -> Capa 3 (platforms)
- **Visual**: Triangulo geometrico minimalista, tres lineas de tag en monospace, demos contextuales por capa
- **Paleta**: Blanco puro + negro, tipografia geometrica + monospace

### 4. Notion — notion.so
- **Metafora**: Sesion musical — "Where teams and agents Jam together"
- **Tecnica**: Colaboracion 24/7, personajes ilustrados, producto como protagonista
- **Arco**: Concepto musical (Jam) -> Colaboracion (teams + agents) -> Automatizacion (24/7) -> Conocimiento (search) -> Confianza (stats)
- **Visual**: Badge "Jam" en purpura, caracteres ilustrados cartoon, interfaz de Ramp HQ, cards de agents con iconos, paleta calida (beige + acentos)
- **Paleta**: Beige/crudo + acentos coloridos (amarillo, azul, naranja, purpura)

### 5. Supabase — supabase.com
- **Metafora**: Transformacion por contraste — "Build in a weekend. Scale to millions"
- **Tecnica**: Contraste visual (negro -> verde), grid modular de features
- **Arco**: Exposicion (construye rapido) -> Desarrollo (herramientas) -> Climax (escala a millones) -> Resolucion (templates)
- **Visual**: Gradiente tipografico negro->verde, grid de 7 features, iconos de frameworks, 104.8K GitHub stars
- **Paleta**: Blanco + verde (#3ECF8E)

## Parte 2: Storytelling Web Inmersivo (5 ejemplos)

### 6. The Sea We Breathe — theseawebreathe.com
- **Experiencia**: Viaje submarino educativo con 3 ecosistemas marinos, narrado por Stephen Fry
- **Tecnica**: WebGL/Three.js, ejercicio de respiracion previo, gestos para resolver amenazas ambientales
- **Estructura**: Hook (respiracion) -> 3 Journeys (Carbon Blue, Protected Areas, Sustainable Fishing) -> Resolution (CTA)
- **Premios**: Awwwards SOTD (7.84), Webby Award

### 7. Into the Storm — intothestorm.io
- **Experiencia**: Documental interactivo de mision de pararescate con CGI cinematografico
- **Tecnica**: WebGL/CGI, soundscape 3D, scroll control de progresion de mision
- **Estructura**: Hook (tormenta) -> Despliegue -> Vuelo -> Salto -> Rescate -> Resolution
- **Premios**: Awwwards SOTD (8.15) + DEV Award — puntuacion mas alta

### 8. Sakharov Space — sakharov.space
- **Experiencia**: Museo digital biografico con 9 capitulos narrativos
- **Tecnica**: WebGL (retrato 3D) + Webflow, simbolismo de luz/sombra
- **Estructura**: Preloader (cita) -> 9 Capitulos (infancia -> bomba H -> activismo -> Nobel -> legado) -> Biblioteca
- **Premios**: Awwwards SOTD + DEV Award

### 9. Black Dog — blackdogstory.com
- **Experiencia**: Libro ilustrado WebGL interactivo en blanco y negro
- **Tecnica**: WebGL/Three.js, transiciones geometricas cinematograficas, efecto de texto borroso
- **Estructura**: Apertura -> Escena 1 (familia) -> Escena 2 (reacciones) -> Escena 3 (revelacion) -> Climax -> Resolution
- **Paleta**: Solo 2 colores (#000000 y #FFFFFF) — restriccion creativa

### 10. Life in Vogue — lifeinvogue.vogue.it
- **Experiencia**: Espacios 3D de moda con transiciones de pagina tridimensionales
- **Tecnica**: Three.js/WebGL, espacios virtuales explorables, eventos en vivo integrados
- **Estructura**: Entrada -> Espacio 1 (Backstage) -> Espacio 2 (Arte) -> Espacio 3 (Eventos) -> Espacio 4 (Cultura) -> Conclusion
- **Premios**: Awwwards SOTD + DEV Award

## Patrones Comunes Identificados (Todos los ejemplos)

1. **Metafora central fuerte** que guia toda la experiencia
2. **Progresion narrativa clara**: Hook -> Desarrollo -> Climax -> Resolution
3. **Scroll como palanca narrativa** (no solo navegacion)
4. **Transiciones como lenguaje narrativo**
5. **Animaciones con proposito** (no decorativas)
6. **Prueba social integrada** (metricas, logos, casos)
7. **Producto como protagonista** (show, don't tell)
