# Plan: Proyecto Hermes - Hackathon Website

## Contexto
El usuario necesita construir un sitio web de dos paginas para un hackathon (Nous Research, Envidia, Stripe):
- **Pagina 1 (Marketing/Branding)**: Explica el concepto del "Partenon" - sistema de agentes (heroes) liderados por Hermes (la empresa), con storytelling usando arquetipos, metricas de crecimiento, estrategia de alcance (universidades, coworking, aceleradoras)
- **Pagina 2 (Desarrolladores/Tecnica)**: Arquitectura tecnica completa, diagramas de flujo, especificaciones tecnicas, conexiones MCP, perfiles tecnicos de cada agente

## Fase 1: Investigacion de Referencias (10+ ejemplos)
**Objetivo**: Encontrar ejemplos de websites con storytelling excepcional, divididos en:
- 5 sitios de documentacion/tecnica con storytelling de marca (ej: Stripe, Linear, Vercel)
- 5 sitios de narrativa inmersiva/experiencias scroll-driven

**Agentes**:
- Agente_Investigador_Marca: Busca sitios de documentacion tecnica con storytelling
- Agente_Investigador_Narrativa: Busca sitios de narrativa inmersiva interactiva

## Fase 2: Diseno y Extraccion de Assets
**Objetivo**: Analizar los mejores ejemplos, extraer bounding boxes de assets visuales que no puedan generarse por codigo (fotografia, ilustraciones organicas)

## Fase 3: Construccion del Sitio Web
**Objetivo**: Construir las dos paginas usando vibecoding-webapp-swarm
- Pagina 1: Partenon - marketing storytelling
- Pagina 2: Arquitectura tecnica para desarrolladores

## Fase 4: Deploy
**Objetivo**: Desplegar el sitio web
