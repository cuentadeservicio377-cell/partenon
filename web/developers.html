Hermes

Recibe intenciones del usuario y las convierte en misiones. Publica en una cola compartida que los perfiles pueden tomar.

G-Brain

Memoria persistente vía MCP. Almacena perfiles, misiones pasadas, preferencias y resultados para mantener contexto entre agentes.

Perfiles

Agentes con habilidades específicas. Cada uno lee su archivo de perfil correspondiente antes de actuar.

Flujo

Cómo se ejecuta una misión.

Perfiles técnicos

Cada perfil, una especificación.

01

Tesorero · .finance

Habilidades
Google Sheets, modelos financieros, costos fijos/variables, dashboards.
Inputs
Transacciones, facturas, estados de cuenta, respuestas del emprendedor.
Outputs
Spreadsheet limpio, categorización, reportes de flujo y proyecciones.
Conexiones
Mensajero (presupuestos), Cobrador (ingresos), G-Brain.
02

Mensajero · .design

Habilidades
Redes sociales, SEO/GEO, copywriting, Open Design, WordPress/SSH.
Inputs
Voz de marca, objetivos de campaña, datos del Tesorero, accesos web.
Outputs
Calendario de contenido, posts, presentaciones, optimizaciones web.
Conexiones
Tesorero (presupuesto), G-Brain, APIs de redes sociales.
03

Cobrador · Stripe

Habilidades
Payment links, suscripciones, facturas, webhooks, reportes.
Inputs
Productos/servicios, precios, clientes, eventos de pago.
Outputs
Links de pago, confirmaciones, reportes de ingresos, reconciliación.
Conexiones
Stripe API, Tesorero (ingresos), G-Brain.
04

Guardián · .security

Habilidades
Gestión de API keys, permisos por perfil, rotación de credenciales.
Inputs
Cuentas de Nvidia, OpenAI, Kimi, Stripe, redes sociales.
Outputs
Accesos configurados, auditoría, políticas mínimas de seguridad.
Conexiones
Nvidia APIs, OpenAI, Kimi, todos los perfiles vía G-Brain.
05

Estratega · Ops

Habilidades
Calendar, Gmail, tareas, recordatorios, priorización.
Inputs
Eventos, correos, deadlines, misiones pendientes.
Outputs
Calendario actualizado, follow-ups, resumen de operaciones.
Conexiones
Diplomático (hitos), G-Brain, Google Workspace.
06

Diplomático · Relaciones

Habilidades
CRM ligero, seguimiento a clientes y proveedores, hitos.
Inputs
Conversaciones, contratos, entregables, fechas compromiso.
Outputs
Recordatorios, actualizaciones de estado, resúmenes de relación.
Conexiones
Estratega (operaciones), G-Brain, Gmail.
Memoria

G-Brain vía MCP.

MCP (Model Context Protocol) es el cable entre los perfiles y el cerebro. Cada agente puede leer contexto, escribir resultados y consultar lo que otros perfiles aprendieron.

  • 01Perfiles: archivos .finance, .design y .security definen comportamiento.
  • 02Misiones: cada tarea queda registrada con input, output y aprendizaje.
  • 03Entidades: clientes, proveedores y productos se guardan como objetos reutilizables.

Workshop técnico

Paquete de implementación de 90 minutos.

0-20'

  • • Pre-instalación con curl
  • • Copiar Google Sheet base
  • • Conectar Google Workspace
  • • Subir .finance, .design, .security
20-60'

  • • Definir tipo de empresa
  • • Configurar perfiles prioritarios
  • • Ejecutar primera misión
  • • Conectar Stripe y redes sociales
60-90'

  • • Publicar 3 misiones reales
  • • Revisar entregables en Sheets/Docs
  • • Ajustar prompts y permisos
  • • Dejar plan de 30 días
Instalación

Empieza en minutos.

1. Clona el repositorio

git clone https://github.com/paolameneses/partenon.git
cd partenon

2. Instala Hermes

curl -fsSL https://partenon.dev/install.sh | bash

El instalador configura el entorno, dependencias y el comando global.

3. Copia el Google Sheet base

El template incluye pestañas para misiones, entidades, perfiles y logs.

4. Configura variables de entorno

HERMES_FOLDER_ID=...
GOOGLE_SERVICE_ACCOUNT=...
STRIPE_API_KEY=...
OPENAI_API_KEY=...
KIMI_API_KEY=...
NVIDIA_API_KEY=...

5. Despliega Apps Script

Sube apps-script/Hermes.gs como proyecto vinculado a tu Google Sheet. Autoriza los scopes de Sheets, Docs, Drive, Calendar y Gmail.
