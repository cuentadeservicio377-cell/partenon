1. Interfaz conversacional

El dueño interactúa por Telegram, WhatsApp o dashboard web. No hay menús: habla como con un socio. Hermes Agent enruta la intención al skill correcto.

2. Hermes Agent Core

El núcleo de enrutamiento, memoria y ejecución de skills. Toma el mensaje del usuario, consulta contexto y decide qué héroe debe actuar.

3. Partenon Profiles

Cada héroe es un skill (o conjunto de skills) con un perfil, herramientas y MCPs. Toman misiones del Partenón y entregan resultados en el workspace.

4. G-Brain como cerebro

Todos los agentes se conectan a G-Brain vía MCP. Ahí vive la memoria compartida, el conocimiento de la empresa y las instrucciones de cada perfil.

5. Google Workspace como superficie

Los datos se escriben en Google Sheets. Los entregables en Docs y Slides. El conocimiento en Drive. La empresa conserva el control y trabaja en herramientas que ya conoce.

6. Stripe para cobranza

El Cobrador se conecta a Stripe API para generar links de pago, suscripciones, facturas y reportes de ingresos integrados con el flujo de ventas.

Flujo

De una idea a un spreadsheet.

Ejemplo: el dueño pregunta cuánto ganó este mes. El Tesorero toma la misión, lee Google Sheets, consulta G-Brain y devuelve un dashboard. 

sequenceDiagram
    actor Owner as Dueño
    participant Hermes as Hermes Agent
    participant Router as Context Router
    participant GBrain as G-Brain
    participant Hero as Tesorero
    participant Sheets as Google Sheets

    Owner->>Hermes: "Quiero saber cuánto gané este mes"
    Hermes->>Router: route(intent)
    Router->>GBrain: consultar contexto empresa
    GBrain-->>Router: perfil + archivos .finance
    Router->>Hero: hermes-finanzas
    Hero->>Sheets: leer / escribir datos
    Sheets-->>Hero: movimientos actualizados
    Hero->>GBrain: guardar análisis
    Hero-->>Hermes: resumen + gráfico
    Hermes-->>Owner: "Ganaste $X. Aquí está tu dashboard."

Workshop técnico

Instalamos Hermes en 90 minutos.

Un formato único para universidades, empresarios, coworkings y aceleradoras. Pre-instalación, onboarding en vivo y primera misión.

1

Pre-instalación

Antes del evento, cada participante corre el comando de instalación y configura su Google Workspace. Llegan con Hermes listo para encender.

2

Onboarding en vivo

Configuramos client.yaml, conectamos Telegram/WhatsApp, validamos el health check y sincronizamos perfiles con G-Brain.

3

Primera misión

Cada participante publica una misión real. Un héroe la toma, actúa y entrega el resultado en Google Sheets antes de terminar la sesión.

Requisitos previos

  • Cuenta de Google Workspace (la gratuita funciona)
  • Bot de Telegram o cuenta de WhatsApp Business
  • Docker instalado
  • API key de modelo (OpenRouter recomendado)
  • Cuenta de Stripe (opcional, para cobranza)
Instalación

Empieza en 5 minutos.

Paso 1 — Instalar Hermes Agent

$ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

Paso 2 — Instalar Partenon skills

$ git clone https://github.com/cuentadeservicio377-cell/hermes-business-os.git
$ cd hermes-business-os
$ bash scripts/install.sh

Paso 3 — Configurar empresa

$ cp config/client.yaml.example config/client.yaml
$ cp .env.example .env
$ docker-compose up -d
