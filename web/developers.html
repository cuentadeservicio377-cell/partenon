1. Interfaz conversacional

El dueño interactúa por Telegram, WhatsApp o dashboard web. No hay menús: habla como con un socio. Hermes Agent enruta la intención al skill correcto.

2. Perfiles de Partenon

Cada héroe es un skill (o conjunto de skills) con un perfil, herramientas y MCPs. Toman misiones del Partenón y entregan resultados en el workspace de la empresa.

3. G-Brain como cerebro

Todos los agentes se conectan a G-Brain vía MCP. Ahí vive la memoria compartida, el conocimiento de la empresa y las instrucciones que cada perfil necesita.

4. Google Workspace como superficie

Los datos se escriben en Google Sheets. Los documentos en Docs/Slides. La empresa conserva el control y trabaja en herramientas que ya conoce.

Integraciones

Conectado a lo que ya usas.

Google Workspace

Sheets como fuente de verdad. Docs y Slides como entregables. Drive como repositorio compartido.

  • • Sheets API (datos maestros)
  • • Docs API (documentos)
  • • Slides API (presentaciones)
  • • Drive API (archivos)
  • • Calendar + Gmail

Stripe

Pagos, suscripciones, recordatorios y reportes de ingresos integrados con el flujo de ventas.

  • • Payment Links
  • • Subscriptions
  • • Invoices
  • • Webhooks
  • • Reporting

G-Brain

Cerebro central del Partenón. Memoria persistente, contexto compartido y orquestación entre agentes.

  • • MCP server
  • • Memoria por empresa
  • • Instrucciones de perfiles
  • • Audit log
  • • Conocimiento compartido

Workshop técnico

Instalamos Hermes en 90 minutos.

1

Pre-instalación

Los participantes corren el comando de instalación y configuran Google Workspace antes del evento.

2

Onboarding en vivo

Configuramos el client.yaml, conectamos Telegram/WhatsApp y validamos el health check.

3

Primera misión

Cada participante publica una misión real y ve cómo un héroe la toma y entrega en Google Sheets.

Requisitos previos

  • • Cuenta de Google Workspace (gratuita funciona)
  • • Bot de Telegram o cuenta de WhatsApp Business
  • • Docker instalado
  • • API key de modelo (OpenRouter recomendado)
  • • Stripe account (opcional, para cobranza)

Instalación

Empieza en 5 minutos.

Paso 1 — Instalar Hermes Agent

$ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

Paso 2 — Instalar Partenon skills

$ git clone https://github.com/cuentadeservicio377-cell/hermes-business-os.git
$ cd hermes-business-os
$ bash scripts/install.sh

Paso 3 — Configurar empresa

$ cp config/client.yaml.example config/client.yaml
$ cp .env.example .env
$ docker-compose up -d
