THE SCRIBE

Guardian of the Ledger

HERO PROFILE

THE SCRIBE

Live Connect Roadmap

The Scribe is the finance and records profile. It parses finance data, classifies expenses, and builds local dashboards and .finance files. Google Sheets and Workspace writes need credentials; Google Drive operations are roadmap.

Parse & classify finance data

Reads uploaded spreadsheets or CSVs, classifies fixed and variable costs, and outputs structured .finance files.

Local dashboards

Builds break-even, margin, and KPI dashboards from local data without external credentials.

Google Workspace writes

Creates and updates Google Sheets when a service account or OAuth credentials are configured.

Local files .finance Files Pivot Tables Query Functions Data Visualization Forecasting Google Sheets

A cafe owner needs to understand their true profitability. The Scribe receives their expense data, creates a .finance file categorizing all costs (rent, supplies, labor, marketing), builds a dashboard showing daily break-even point, and alerts when margins drop below target.

THE COLLECTOR

Keeper of Payments

HERO PROFILE

THE COLLECTOR

Live Connect Roadmap

The Collector is the payments profile. It runs dry-run invoices, subscriptions, and revenue reports, and creates Stripe payment links. Live charges require a Stripe secret key; Stripe Connect is roadmap.

Dry-run invoicing

Creates invoice drafts and subscription schedules locally before any live charge.

Revenue reports

Builds MRR, ARR, and churn reports from Stripe data or mock data.

Payment links

Generates Stripe Checkout and payment-link URLs.

Live Stripe execution

Processes real charges, invoices, and webhooks when STRIPE_SECRET_KEY is set.

TOOLS
Stripe API Payment Links Checkout Billing Invoicing Stripe Connect (roadmap)
WORKS CLOSELY WITH
The Scribe
The Strategist
The Brain
MCP CONNECTIONS bolt Stripe integration (connect)

A coworking space sells day passes, monthly memberships, and meeting room bookings. The Collector sets up Stripe Checkout for each product type, configures subscription billing for memberships, creates an automated invoicing system for corporate clients, and builds a revenue dashboard.

CENTRAL INTELLIGENCE

THE BRAIN

G-BRAIN OF GARRY TAN

Live Connect Roadmap

The Brain is shared memory. It stores memory pages and profiles, runs semantic search, and detects conflicts. It uses SQLite by default and can scale to Postgres when connected.

Memory pages

Stores structured notes, decisions, and context across sessions.

Profile storage

Keeps hero and enterprise profile data consistent.

Conflict detection

Flags contradictions and duplicates in shared memory.

SQLite / Postgres

Runs locally on SQLite; connect GBRAIN_DATABASE_URL for team Postgres.

bolt MCP POWERED

Model Context Protocol enables The Brain to connect every hero to shared context, tools, and memory — creating a true collective intelligence.

HOW THE PARTENON RUNS

Four steps from installation to autonomous operation. Everything starts in dry-run mode; connect credentials only when you are ready for live integrations.

route FROM INSTALL TO AUTONOMY
download
STEP 1 — Install

Download the Partenon

Clone the repository and run ./install.sh. The FastAPI backend, local memory store, and dashboard start immediately. Hermes is an optional dependency for the CLI profile experience.

check_circle No credentials required to start
STEP 2 — Configure

Assemble Your Hero Ensemble

Pick the profiles your enterprise needs. Each one runs locally out of the box. Add API keys or service accounts only for the integrations you want to activate.

check_circle Mix live, connect, and roadmap profiles
STEP 3 — Operate

Missions in Motion

The Strategist routes tasks; each hero executes its domain. Every integration defaults to dry-run until you explicitly set dry_run: false and supply valid credentials.

check_circle Safe by default, live on demand
STEP 4 — Orchestrate

The Partenon Drives

The Brain stores shared memory, detects conflicts, and keeps every hero aligned. As more integrations connect, the ensemble becomes increasingly autonomous.

check_circle Cross-agent memory and continuity
live = works without credentials connect = needs your API key or account roadmap = not yet implemented
