Sistema

No es mitología. Es un sistema de arquetipos operativos.

Las empresas pequeñas no fallan por falta de esfuerzo. Fallan porque las finanzas, la comunicación, los pagos, la seguridad, las operaciones y las relaciones viven en silos. Partenon modela esos seis roles como perfiles de agente con herramientas reales.

[ 01 ]

Hermes es la empresa

No es un CEO. No es un chatbot. Hermes es el nombre del sistema que representa a la empresa: publica misiones, prioriza y decide qué necesita.

[ 02 ]

Los héroes son perfiles

Cada perfil tiene un archivo de instrucciones y un set de herramientas. Tesorero, Mensajero, Cobrador, Guardián, Estratega y Diplomático no son personajes. Son funciones operativas.

[ 03 ]

El Partenón es el workspace

Todos trabajan en el mismo lugar: Google Workspace. Sheets, Docs, Drive, Calendar y Gmail son la superficie donde la empresa y los agentes colaboran.

Hermes publica. Los perfiles ejecutan. Tu workspace recibe.

Una misión empieza cuando la empresa describe un problema. Hermes la descompone. El perfil adecuado la toma, lee su archivo de instrucciones, consulta al cerebro compartido y entrega el resultado donde el emprendedor ya trabaja.

Herramientas

Cada perfil tiene su Pegaso.

Como Hércules tenía su caballo, cada perfil tiene herramientas open source y habilidades que le permiten cumplir misiones reales. No son adornos. Son conectores probados.

FILE
.finance

Instrucciones de finanzas, costos fijos/variables, dashboards y reportes.

FILE
.design

Voz de marca, calendario de contenido, SEO/GEO, accesos web.

FILE
.security

API keys, permisos por perfil, modelos y credenciales.

INTEGRATION
MCP

G-Brain conecta memoria, perfiles y misiones entre todos los agentes.

Perfiles operativos

Seis funciones. Una mesa de trabajo.

01

Tesorero

Ordena números. Carga el archivo .finance, lee los datos de la empresa y construye dashboards en Google Sheets.

CASO A Empresa de construcción

Materiales, mano de obra, margen por obra, costos fijos y variables.

CASO B Cafetería

Pauta de marketing, impuestos, renta, insumos, nómina.

Google Sheets Dashboards .finance Costos fijos/variables
02

Mensajero

Ventas y comunicación. Carga .design, investiga la marca con el emprendedor y ejecuta campañas, calendarios de contenido, SEO/GEO y presentaciones.

MISIONES DE INVESTIGACIÓN
¿Qué vendes? ¿A quién ayudas? ¿Por qué te necesitan? ¿Cómo lo resuelves?
Open Design SEO / GEO WordPress / SSH Google Sheets
03

Cobrador

Recibe pagos por Stripe. Crea payment links, suscripciones, facturas y reportes de ingresos. Conecta tiendas en línea, servicios y productos físicos.

Stripe API Payment links Suscripciones Reportes de ingresos
04

Guardián

Seguridad y modelos. Carga .security, administra API keys, permisos por perfil y accesos a Nvidia, OpenAI, Kimi Coding y cuentas pagadas.

Nvidia APIs OpenAI Kimi Coding API keys
05

Estratega

Administración y project management. Gestiona Calendar, Gmail, recordatorios, hitos y detalles de operaciones. Es el gerente operativo interno.

Google Calendar Gmail Tareas Recordatorios
06

Diplomático

Relaciones con clientes y proveedores. Registra hitos, envía recordatorios y media entre lo que pide el cliente y lo que puede entregar operaciones.

Funciona como el punto medio entre dos frentes: sabe lo que el cliente necesita y lo que operaciones puede entregar. Su trabajo es que nada se pierda en la negociación.

Clientes Proveedores Hitos Recordatorios
Memoria

Sin cerebro, los perfiles no se hablan.

G-Brain de Garitán es la memoria compartida. Cada perfil puede leer contexto, escribir resultados y consultar lo que otros perfiles aprendieron. La conexión usa MCP.

  • 01 Perfiles: archivos .finance, .design y .security definen comportamiento.
  • 02 Misiones: cada tarea queda registrada con input, output y aprendizaje.
  • 03 Entidades: clientes, proveedores y productos se guardan como objetos reutilizables.

Superficie común

Todo pasa en Google Workspace.

Si Hermes trabaja solo con archivos Markdown, los empresarios no tienen nada que tocar. Partenon ancla todo a Google Workspace: Sheets, Docs, Slides, Drive, Calendar y Gmail. Funciona con una cuenta gratuita.

  • Los resultados viven donde ya trabajan.
  • Los perfiles comparten archivos y contexto.
  • El emprendedor puede auditar todo.
