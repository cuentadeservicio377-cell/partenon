# Partenon Onboarding Checklists by Business Type

These checklists tailor the generic onboarding flow to five common small-business types.

---

## Coffee shop / cafe

### Week 1: Numbers first

- [ ] Run the Scribe demo and confirm the margin report format.
- [ ] Create `.finance` with rent, payroll, supplies, and COGS buckets.
- [ ] Import or manually enter last month's POS sales and supplier invoices.
- [ ] Classify every cost as fixed or variable and identify the biggest variable spend.

### Week 2: Operations rhythm

- [ ] Create `.ops` with morning briefing, midday pulse, and weekly planning.
- [ ] Build a weekly prep checklist: inventory, equipment, staffing, cleaning.
- [ ] Add supplier delivery cutoff reminders to the Strategist calendar.

### Week 3: Marketing and payments

- [ ] Complete `.design` with brand voice, channels, and claims to avoid.
- [ ] Generate a 14-day Instagram/Reels calendar.
- [ ] Create Stripe payment links for catering and loyalty subscriptions.

### Week 4: Review

- [ ] Run a weekly budget review and flag any category over budget by >10%.
- [ ] Review low-margin days and adjust scheduling.
- [ ] Add Guardian to audit shared POS/social logins.

---

## Marketing / professional services agency

### Week 1: Projects and clients

- [ ] Create `.ops` with retainer project template (discovery, strategy, creative, launch, optimization, reporting).
- [ ] Register all active clients and prospects in `.relations`.
- [ ] Add milestones for proposal delivery, kickoff, and monthly QBR.

### Week 2: Profitability

- [ ] Create `.finance` with office rent, software stack, freelance/media-buyer budgets, and ad-spend pass-through.
- [ ] Track retainer revenue vs. estimated delivery hours per client.
- [ ] Flag any account being over-serviced.

### Week 3: Pipeline and content

- [ ] Set up Diplomat daily follow-ups for milestones in the next 3 days.
- [ ] Complete `.design` and generate case-study copy from recent wins.
- [ ] Create a 30-day content calendar for LinkedIn and newsletter.

### Week 4: Billing

- [ ] Create Collector invoices for monthly retainers.
- [ ] Set up overdue-account reminders.
- [ ] Generate the income report and review days-sales-outstanding.

---

## Construction / events / logistics

### Week 1: Phases and safety

- [ ] Create `.ops` with phase checklists (pre-construction, construction, closeout).
- [ ] Register all clients, subcontractors, and suppliers in `.relations`.
- [ ] Add milestones for permits, inspections, material deliveries, and punch-list sign-off.

### Week 2: Project costing

- [ ] Create `.finance` with fixed costs (rent, insurance, payroll) and variable budgets (materials, subcontractor labor).
- [ ] Create a project cost workbook per active job.
- [ ] Compare actual spend to estimate weekly.

### Week 3: Security and access

- [ ] Create `.security` with provider entries for supplier portals, CAD tools, and insurance systems.
- [ ] Run Guardian audit and flag shared logins.
- [ ] Set 90-day rotation reminders.

### Week 4: Billing and closeout

- [ ] Create Collector progress-payment invoices.
- [ ] Set up reminders for retainers and overdue accounts.
- [ ] Index project lessons in the Brain for future estimates.

---

## Retail / e-commerce

### Week 1: Inventory and margins

- [ ] Create `.finance` with rent, payroll, and COGS by category (books, gifts, events, online).
- [ ] Import last month's sales and cost data.
- [ ] Identify the highest- and lowest-margin categories.

### Week 2: Payments and orders

- [ ] Create `.payments` with products for in-store, online, event ticket, and subscription sales.
- [ ] Create Stripe payment links or checkout products.
- [ ] Set up refund and failed-payment policies.

### Week 3: Marketing calendar

- [ ] Complete `.design` with brand voice and channels.
- [ ] Generate a 30-day promotional/event calendar.
- [ ] Tie campaigns to restock dates in the Strategist.

### Week 4: Supplier relationships

- [ ] Register top distributors and suppliers in `.relations`.
- [ ] Add milestones for catalog deadlines, allocation releases, and contract renewals.
- [ ] Run Diplomat follow-ups weekly.

---

## SaaS / tech startup

### Week 1: Security and spend

- [ ] Create `.security` with API keys for OpenAI, Stripe, cloud provider, and any model providers.
- [ ] Run Guardian audit and flag missing or expired keys.
- [ ] Create `.finance` with cloud, software, payroll, and model-API budgets.

### Week 2: Subscriptions and unit economics

- [ ] Connect or simulate Stripe products and subscriptions in `.payments`.
- [ ] Generate the monthly income report and review failed payments/refunds.
- [ ] Build a unit-economics dashboard: MRR, cloud spend, support cost per customer.

### Week 3: Product execution

- [ ] Create `.ops` with sprint projects and incident post-mortem templates.
- [ ] Assign top support themes as projects with owners and due dates.
- [ ] Schedule weekly retrospectives.

### Week 4: Memory and communication

- [ ] Index validated decisions in `.brain` (pricing, model selection, refund policy, architecture).
- [ ] Complete `.design` for changelog and founder updates.
- [ ] Generate a 14-day content calendar.
