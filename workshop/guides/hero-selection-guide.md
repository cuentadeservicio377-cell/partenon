# Partenon Hero Selection Guide

Use this guide to decide which heroes to activate first for a given business type. The order is based on where the most friction sits in the first 30 days.

---

## Quick reference

| Business type | Activate first | Why |
|---------------|----------------|-----|
| Coffee shop / cafe | Scribe → Strategist → Herald → Collector | Tight margins, recurring operations, local marketing. |
| Marketing / professional services agency | Strategist → Diplomat → Scribe → Herald → Collector | Deadlines, client follow-ups, retainer profitability. |
| Construction / events / logistics | Strategist → Diplomat → Scribe → Guardian | Phases, suppliers, physical deadlines, safety. |
| Retail / e-commerce | Scribe → Collector → Herald → Strategist | Inventory, payments, promotions, seasonality. |
| SaaS / tech startup | Guardian → Scribe → Collector → Strategist → Brain | Security, unit economics, subscriptions, decisions. |

---

## Coffee shop / cafe

**Typical pains**: labor scheduling, waste, supplier payments, local marketing, thin margins.

**Hero order**

1. **Scribe** — classify rent, payroll, supplies, and daily sales into fixed/variable costs; build a margin dashboard.
2. **Strategist** — schedule staff, suppliers, and social posts; remind the owner of low-margin days.
3. **Herald** — generate weekly Instagram/Reels calendars and local SEO copy.
4. **Collector** — set up Stripe payment links for catering orders and loyalty subscriptions.

Add Guardian in month two for shared POS/social logins.

---

## Marketing / professional services agency

**Typical pains**: scope creep, proposal stalls, client communication, multi-project reporting.

**Hero order**

1. **Strategist** — track projects, tasks, deadlines, and resource allocation.
2. **Diplomat** — log client milestones, contracts, and follow-ups.
3. **Scribe** — match project budgets to actual hours and vendor costs.
4. **Herald** — generate case-study content and proposal copy.
5. **Collector** — invoice retainers and follow up on overdue accounts.

---

## Construction / events / logistics

**Typical pains**: subcontractor coordination, schedule delays, change orders, cash flow, permits.

**Hero order**

1. **Strategist** — build phase checklists (pre-construction, construction, closeout).
2. **Diplomat** — track suppliers, contracts, and confirmation milestones.
3. **Scribe** — cost each project separately: materials, labor, equipment, subcontractors.
4. **Guardian** — protect shared accounts for supplier portals, CAD tools, and insurance systems.

Add Collector once progress billing is ready.

---

## Retail / e-commerce

**Typical pains**: inventory accuracy, stockouts/overstock, payment reconciliation, promotions, seasonality.

**Hero order**

1. **Scribe** — track COGS, shipping, ads, and margins by product line.
2. **Collector** — handle Stripe checkout, subscriptions, and refunds.
3. **Herald** — plan promotional calendars and product launches.
4. **Strategist** — coordinate restocking, campaigns, and supplier deadlines.

Add Diplomat for distributor and corporate-account relationships.

---

## SaaS / tech startup

**Typical pains**: API key hygiene, cloud spend, subscription billing, sprint prioritization, knowledge transfer.

**Hero order**

1. **Guardian** — audit API keys, model access, and provider accounts from day one.
2. **Scribe** — track cloud costs, contractor spend, and runway.
3. **Collector** — manage Stripe subscriptions, trials, and churn signals.
4. **Strategist** — run sprints, investor updates, and hiring pipelines.
5. **Brain** — index decisions about pricing, positioning, and architecture.

Add Herald for changelog and founder updates.

---

## When to call a human

Partenon heroes draft, organize, and remind. They do not replace professional judgment:

- Taxes and legal contracts → accountant / lawyer signs off.
- Security incidents → security engineer investigates.
- Client disputes → owner makes the final call.
- Regulated advice → never delegate to an agent.
